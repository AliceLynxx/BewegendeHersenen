# BewegendeHersenen - Project Informatie

## Projectomschrijving
BewegendeHersenen is een Python library die het eenvoudig maakt om dynamische animaties te creëren met matplotlib op basis van numpy arrays. Het project richt zich op het visualiseren van bewegende patronen, data-evolutie over tijd, en interactieve visualisaties.

## Project doel, nut en functie
**Doel**: Een gebruiksvriendelijke Python library ontwikkelen waarmee gebruikers snel en intuïtief animaties kunnen maken van hun numerieke data.

**Doelgroep**: 
- Data scientists die hun analyses willen visualiseren
- Onderzoekers die bewegende patronen willen tonen
- Studenten en docenten voor educatieve doeleinden
- Ontwikkelaars die animaties nodig hebben in hun projecten

**Nut**: 
- Vereenvoudigt het proces van animatie-creatie in Python
- Biedt herbruikbare templates voor veelvoorkomende animatietypes
- Maakt complexe data-visualisatie toegankelijk voor beginners
- Bespaart tijd door gestandaardiseerde animatiefuncties

**Functie**:
- Transformeert numpy arrays naar vloeiende matplotlib animaties
- Biedt verschillende animatietypes (lijngrafiek, scatter, heatmap, etc.)
- Ondersteunt export naar verschillende formaten (GIF, MP4, HTML)
- Configureerbare animatiesnelheid en styling
